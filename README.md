# 📘 Champion des Maths

## 🎮 Description

**Champion des Maths** est un jeu éducatif conçu pour les élèves de niveau **CM1**. Il permet de s’entraîner aux calculs mathématiques tout en s’amusant. Le joueur progresse à travers différents jeux en résolvant des opérations adaptées à son niveau scolaire.

## 🎯 Objectifs pédagogiques

* Renforcer les compétences en calcul mental
* Maîtriser les opérations de base : addition, soustraction, multiplication et division
* Développer la rapidité et la logique
* Donner confiance en mathématiques grâce au jeu

## 🧠 Contenu du jeu

Le jeu propose plusieurs types d’exercices :

* ➕ Additions
* ➖ Soustractions
* ✖️ Multiplications
* ➗ Divisions simples
* 🔢 Tables de multiplications

## 🕹️ Fonctionnement

* Le joueur répond à des questions mathématiques
* Chaque bonne réponse rapporte des points ⭐
* Les mauvaises réponses font baisser le nombre de point gagnés


## 👨‍🏫 Public cible

* Élèves de **CM1**
* Enseignants souhaitant un support ludique
* Parents accompagnant leurs enfants dans l’apprentissage
